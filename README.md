# Quant Analysis

This is a collection of scripts and notebooks for analyzing the quant data.

## Setup

### Install dependencies

```bash
pip install -r requirements.txt
```

### Download data

```bash
python download_data.py
```

### Run notebooks

```bash
jupyter notebook
```
